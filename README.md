# Qualbot — Live-Config Backtest Reports

Interactive backtest reports for a long-only momentum-breakout swing strategy (Qullamaggie-style),
deployed live on Hyperliquid (crypto + TradFi perps) and Alpaca (US equities).

- **[Alpaca — US equities](https://boraodbsi.github.io/qualbot-backtests/alpaca-live-baseline/)**
- **[Hyperliquid — crypto + TradFi](https://boraodbsi.github.io/qualbot-backtests/hyperliquid-live-baseline/)**

Click any trade in a report to open its entry/exit candlestick chart.
Period 2023-01-01 → 2026-06-19. Survivorship-bias-controlled (point-in-time universe snapshots).
Costs modeled: 0.15% breakout slippage + 0.035% commission. Backtested, not live, results.
